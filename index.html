Secure Text Encryption

AES-256-GCM End-to-End Encryption

Store your password securely. Use a password manager. Data is cleared automatically after 5 minutes of inactivity. Press ESC to clear all data immediately.

How It Works

  • Encryption: Uses AES-256-GCM for military-grade security
  • Password-based: Your password generates a unique encryption key using PBKDF2 (600,000 iterations)
  • Cross-platform: Share encrypted text between any device
  • Private: Everything happens in your browser, no data is sent to servers

Security Features

  • Auto-clear: All sensitive data is automatically cleared after 5 minutes of inactivity
  • Secure memory wipe: Password and plaintext are securely wiped from memory
  • Clipboard protection: Copied encrypted text is cleared from clipboard after 60 seconds
  • Password strength: Real-time feedback helps you create strong passwords
  • No autocomplete: Prevents browser from storing sensitive information
  • Keyboard shortcut: Press ESC to instantly clear all data
